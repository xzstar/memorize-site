# 背书练习台

一个零依赖的静态背诵练习网站。输入一段中文或英文后，页面会按句随机减少 10%~20% 的内容，帮助学生用回忆训练替代机械重读。

## 适合谁

- 需要背语文课文、古诗词、定义、法条的学生
- 需要背英语课文、演讲稿、名句的学习者
- 想快速生成练习材料的老师或家长

## 已实现功能

- 中文与英文输入
- 自动识别语言，也支持手动切换中文或英文模式
- 中文按字挖空，英文按词挖空
- 支持两种练习模式：整段练习、闯关训练
- 整段练习支持每句话随机减少 10%~20% 的内容
- 整段练习支持固定比例 `10% / 15% / 20%`
- 支持三种遮挡样式：下划线、方块、圆点
- 支持重新随机，生成不同的练习版本
- 支持显示原文 / 隐藏原文
- 支持复制当前练习结果
- 闯关训练支持按文本长度动态推进：`单句关 -> 双句重叠 -> 整段总关`，长文会额外加入 `三句重叠`；如果原文有明显段落，系统会优先按原段落拆关、显示当前段落进度，再进入整篇总关，并配合 `20% / 40% / 60% / 80%` 难度梯度、手动跳级、失败建议降一级、单题重新随机、错题重练与回合总结；最终整篇总关会有单独的最终挑战提示和通关总结
- 支持中文示例与英文示例
- 输入内容与设置自动保存在浏览器本地
- 桌面端与移动端均可使用

## 界面说明

页面包含三个主要区域：

1. 原文输入区
2. 练习设置区
3. 练习结果区（可切换整段练习 / 闯关训练）

用户粘贴文本后，可以先选择练习模式，再设置语言模式、挖空比例和遮挡样式，最后生成结果。

## 背后规则

- 文本会先做轻量清洗，保留段落结构
- 再按句子切分内容
- 中文句子按单字作为可隐藏单位
- 英文句子按单词作为可隐藏单位
- 标点和空格保留，不参与挖空
- 每句话独立计算隐藏比例与隐藏位置

这样同一段文字每次都能生成不同版本，但整体难度保持稳定。

## 主要交互

- `生成练习`：根据当前文本和设置生成一版新练习
- `重新随机`：保留原文与设置，只更新缺失位置
- `显示原文`：在结果区切换到原文对照状态
- `复制结果`：复制当前可见结果，方便发给同学或打印整理
- `闯关训练`：按“先看原文 -> 开始测试 -> 本题可重新随机 -> 查看答案 -> 标记是否记住”的节奏完成当前关卡训练，并可在 `20% / 40% / 60% / 80%` 间手动切换；长文会继续推进到 `双句重叠 / 三句重叠 / 整段总关`，多段原文会优先拆成分段关卡并显示当前段落进度
- `加载中文示例 / 加载英文示例`：快速体验页面效果

## 当前技术方案

- `HTML`：负责页面结构
- `CSS`：负责响应式布局与视觉风格
- `JavaScript`：负责语言识别、分句、分词、随机挖空、结果渲染和本地存储

项目没有使用构建工具、框架或后端服务，适合快速部署到任意静态托管平台。
